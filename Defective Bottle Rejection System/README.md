# Defective Bottle Inspection System

Automated quality control system designed to detect and remove damaged bottles from a production line.

## Project Overview

The system inspects bottles transported on a conveyor belt and verifies the presence of a bottle neck using optical sensing.

If a defect is detected, a pneumatic actuator removes the bottle from the production line.

## Features

- Bottle detection
- Neck inspection
- Defect identification
- Pneumatic rejection mechanism
- Conveyor control
- Automatic operation

## Technologies

- Siemens LOGO! PLC
- Optical Sensors
- Pneumatic Actuators
