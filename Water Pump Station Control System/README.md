# Water Pump Station Control System

PLC project created in Siemens LOGO! Soft Comfort for automatic control of a water tank filling and draining process.

## Project Overview

The system controls the liquid level inside a storage tank using three level sensors. Depending on the current level, the controller automatically starts filling or draining operations while maintaining stable operation through hysteresis control.

An alarm mode was also implemented to protect the pump against overheating.

## Features

- Automatic tank filling
- Automatic tank draining
- Three-level liquid monitoring
- Hysteresis control
- Pump overtemperature protection
- Audible and visual alarm system
- FBD and LAD implementation

## Technologies

- Siemens LOGO! PLC
- LOGO! Soft Comfort
- FBD (Function Block Diagram)
- LAD (Ladder Diagram)
- GRAFCET
