# Material Sorting Station

PLC-controlled sorting system capable of distinguishing between aluminium, white plastic and black plastic elements.

## Project Overview

The station uses multiple industrial sensors to identify the material type and activate the corresponding pneumatic actuator.

The program ensures that only one actuator can operate at a time, preventing collisions and incorrect sorting operations.

## Features

- Material identification
- Aluminium detection
- Plastic classification
- Pneumatic sorting mechanism
- Sensor-based decision making
- Collision prevention logic

## Technologies
