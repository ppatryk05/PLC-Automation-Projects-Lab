# Dairy Cup Sealing Production Line

Automation project simulating a production line used for sealing dairy cups.

## Project Overview

The system synchronizes a conveyor belt with a pneumatic cylinder responsible for sealing cup lids. Sensors detect the position of the product and trigger the sealing process at the correct moment.

The project also includes emergency stop handling and automatic return to a safe state.

## Features

- Conveyor belt control
- Cup position detection
- Pneumatic actuator control
- Automatic sealing sequence
- Emergency stop procedure
- Sequential process control

## Technologies

- Siemens LOGO! PLC
- LOGO! Soft Comfort
- FBD Programming
- GRAFCET
