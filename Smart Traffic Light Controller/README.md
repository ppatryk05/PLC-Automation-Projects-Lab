# Smart Traffic Light Controller

Traffic light control system developed in Siemens LOGO! Soft Comfort.

## Project Overview

The project simulates a real traffic intersection with support for vehicle traffic, pedestrian requests and night mode operation.

The controller uses timers and state-based logic to manage safe transitions between signal phases.

## Features

- Vehicle traffic control
- Pedestrian crossing support
- Request button handling
- Night mode
- Emergency stop functionality
- Timer-based sequencing

## Technologies

- Siemens LOGO! PLC
- LOGO! Soft Comfort
- FBD
- State Machine Design
- GRAFCET
